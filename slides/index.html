P05

Install Memory

ติดตั้งระบบความจำ (MCP Server)


# Clone oracle-v2

gh repo clone Soul-Brews-Studio/oracle-v2

cd oracle-v2 && bun install


# Add to claude_desktop_config.json

"oracle-v2": {

"command": "bun",

"args": ["run", "start"]

}

P06

Install Skills

ติดตั้ง Oracle Skills


# Clone skills CLI

gh repo clone Soul-Brews-Studio/oracle-skills-cli


# Install globally

bun run install --agent claude-code --global
