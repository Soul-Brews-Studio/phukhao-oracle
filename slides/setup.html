Oracle Setup Guide

วิธีติดตั้ง Oracle ของคุณเอง


Step-by-step technical guide

S01

สิ่งที่ต้องมี


GitHub CLI gh
Bun bun (or npm)
Claude Code or Claude Desktop
S02

1Clone Starter Kit


# Clone the starter kit

gh repo clone Soul-Brews-Studio/oracle-starter-kit


# Enter directory

cd oracle-starter-kit

S03

2The ψ/ Brain Structure


ψ/

├── inbox/ ← Messages, handoffs

├── memory/ ← Knowledge base

│ ├── learnings/ ← Patterns discovered

│ ├── retrospectives/ ← Session records

│ └── resonance/ ← Core identity

├── active/ ← Work in progress

└── lab/ ← Experiments

S04

3Install Memory (MCP)


# Clone oracle-v2 MCP server

gh repo clone Soul-Brews-Studio/oracle-v2

cd oracle-v2 && bun install


MCP = Model Context Protocol

ระบบความจำถาวรของ Oracle

S05

Configure MCP

เพิ่มใน claude_desktop_config.json


{

"mcpServers": {

"oracle-v2": {

"command": "bun",

"args": ["run", "start"],

"cwd": "/path/to/oracle-v2"

}

}

}

S06

4Install Skills


# Clone skills CLI

gh repo clone Soul-Brews-Studio/oracle-skills-cli


# Install globally

cd oracle-skills-cli

bun run install --agent claude-code --global

S07

Skills ที่ได้รับ


Skill Description
/recap สรุป context
/rrr สร้าง retrospective
/trace ค้นหาประวัติ
/forward สร้าง handoff

[Your Oracle's purpose]

S09

6Create Resonance

บุคลิกภาพของ Oracle


# Create identity file

mkdir -p ψ/memory/resonance

touch ψ/memory/resonance/identity.md


เขียนเกี่ยวกับ:

• ชื่อและความหมาย

• สไตล์การสื่อสาร

• ค่านิยมหลัก

S10

7Test Your Oracle

/standup
