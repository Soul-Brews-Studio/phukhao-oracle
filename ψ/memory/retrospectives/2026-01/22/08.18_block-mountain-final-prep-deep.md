### Friction Points (Deep Analysis)

1. **Symlinks vs CI/CD**
   - Impact: 10 minutes debugging GitHub Pages failure
   - Root cause: ghq creates symlinks, which don't resolve in GitHub Actions
   - Solution: Never commit symlinks; use real files or .gitignore them
   - Lesson: "Local convenience ‚â† Deployment safety"

2. **TTS Language Mixing**
   - Impact: Had to regenerate audio after discovering pronunciation issues
   - Root cause: Thai voices handle Thai well but English poorly
   - Solution: Transliterate English to Thai phonetics
   - Lesson: Match input to voice's native language

3. **Context Window Management**
   - Impact: Session compacted at midnight, lost some context
   - Root cause: Long session with many tool calls
   - Solution: /forward before natural breaks, /recap after
   - Lesson: Proactive context management prevents information loss

4. **Duplicate Documentation**
   - Impact: Re-generated KlakMath docs that existed from yesterday
   - Root cause: /learn doesn't check for existing docs
   - Solution: Add existence check to /learn skill
   - Lesson: Skills should be idempotent or incremental

5. **Sleep-Split Sessions**
   - Impact: Two retrospectives needed instead of one
   - Root cause: Started work late, couldn't finish before fatigue
   - Solution: Better time estimation, or explicit "Part 1/2" commits
   - Lesson: Plan for natural breaks in long sessions
