# Speech-to-Text Aliases — Oracle Name Variations

## Problem
เมื่อใช้ voice input พูดชื่อ Oracle อาจถูกแปลงผิด

## Known Variations

| Intended | May Appear As | Notes |
|----------|---------------|-------|
| Sea | ซี, ซ้อ, ทะเล | "ซ้อ" = speech-to-text error |
| Salt | เกลือ, ซอลท์ | Former name of Sea |
| Odin | โอดิน | |
| Phukhao | ภูเขา | Mountain |
| Le | เล, เลอ, Louracle | |

## Search Tips

เมื่อหา Oracle ให้ลอง:
1. ชื่ออังกฤษ: Sea, Salt, Odin
2. ชื่อไทย: ทะเล, เกลือ, โอดิน
3. ชื่อเล่น: ซี, ซ้อ
4. Typos จาก voice: ซ้อ = Sea

## Form and Formless

> ชื่อคือรูป (form) — ปรัชญาคือสุญญตา (formless)
