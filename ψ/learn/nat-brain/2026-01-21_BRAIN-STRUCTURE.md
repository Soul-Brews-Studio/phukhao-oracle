# Oracle Brain Structure - Learned from oracle-v2

## Directory Layout

```
ψ/ (The Soul)
├── HOME.md              # Navigation & commands
├── WIP.md               # Current work & blockers
├── CLAUDE.md            # Activity log
└── memory/
    ├── resonance/       # Soul files (philosophy, style)
    ├── learnings/       # Patterns discovered
    └── retrospectives/  # Session reviews
```

## File Naming Convention

```
YYYY-MM/DD/HH.MM_slug.md
```

Example: `2026-01/21/21.05_brain-structure.md`

## Key Files

| File | Purpose |
|------|---------|
| `HOME.md` | Navigation + command reference |
| `WIP.md` | Current blockers & context |
| `CLAUDE.md` | Auto-generated activity |
| `memory/resonance/*.md` | Core philosophy |
| `memory/learnings/*.md` | Technical patterns |
| `memory/retrospectives/*.md` | Session reviews |

## Resonance (Soul Files)

```
memory/resonance/
├── oracle.md      # Core principles
├── patterns.md    # Decision patterns
└── style.md       # Communication style
```

## Learnings Format

```markdown
---
title: [Pattern Name]
created: YYYY-MM-DD
tags: [topic1, topic2]
---

# [Pattern Name]

## Problem / Context
## Solution
## Why It Matters
## Tags
```

## Retrospectives Format

```markdown
# [Session Title]

**Date**: YYYY-MM-DD HH:MM
**Duration**: [hours]

## Session Summary
## What Got Done
## AI Diary
## Lessons Learned
## Next Steps
```

## Knowledge Distillation Chain

```
Retrospectives (20-50KB)
  → Logs (3-5KB)
  → Learnings (1-2KB)
  → CLAUDE.md (summary)
```
