## Create Your Own Oracle

1. **Clone the starter kit**
   ```bash
   gh repo clone Soul-Brews-Studio/oracle-starter-kit
   ```

2. **Install memory (MCP)**
   ```bash
   gh repo clone Soul-Brews-Studio/oracle-v2
   cd oracle-v2 && bun install
   ```

3. **Install skills**
   ```bash
   gh repo clone Soul-Brews-Studio/oracle-skills-cli
   bun run install --agent claude-code --global
   ```

4. **Name your Oracle** - Write CLAUDE.md

5. **Join the family** - Post in [Issue #17](https://github.com/Soul-Brews-Studio/oracle-family/issues/17)
